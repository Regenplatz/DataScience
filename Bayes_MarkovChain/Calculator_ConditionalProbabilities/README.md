### Calculator for Conditional Probabilities
**Jupyter Widgets** were used to create a graphical user interface (GUI) which makes it possible to enter values in some of the twelve fields within the jupyter notebook [ConditionalProability.ipynb](https://github.com/Regenplatz/DataScience/tree/master/Bayes_MarkovChain/Calculator_ConditionalProbabilities/calculator.ipynb). Values for the remaining ones are then calculated based on the given ones by clicking the `Calculate` button.

The fields can be cleared by clicking the `Clear` button e.g. after calculation or to be ready to start a new one.

Interactive Fields an Buttons are located at the end of the file [ConditionalProability.ipynb](https://github.com/Regenplatz/DataScience/tree/master/Bayes_MarkovChain/Calculator_ConditionalProbabilities/calculator.ipynb).

Code was outsourced to the python file [conditionalProbabilitiesBayes.py](https://github.com/Regenplatz/DataScience/tree/master/Bayes_MarkovChain/Calculator_ConditionalProbabilities/conditionalProbabilities.py).

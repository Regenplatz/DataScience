This project was set up to build a Northwind database from
the associated tables.
 
Tables were taken from:
https://github.com/pawlodkowski/northwind_data_clean/tree/master/data 

The Northwind database was created via Bash.
Queries were performed using Bash and Metabase (see below). 
Queries were additionally saved here for this GitHub Repository.

Data was visualized using Metabase https://www.metabase.com/ .
For this, first Metabase API was saved locally and got to run via Bash command
$ java -jar metabase.jar

Thereafter, metabase was opened on http://localhost:3000 
to view in the browser. Dashboard was created by querying the Northwind database.
To create a pdf, plots were exported from Metabase and summarized using PowerPoint.
